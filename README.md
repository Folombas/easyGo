## 📖 Описание

easyGo — это коллекция простых и наглядных примеров кода на Go для начинающих. Каждый пример оформлен красочно, с эмодзи и понятными объяснениями.

**Особенности:**
- 🎨 Красочный вывод с использованием `fatih/color`
- 📚 Пошаговые уроки
- 💡 Понятные примеры
- 🚀 Быстрый старт
- ✨ Минимум теории, максимум практики

## 🏃 Быстрый старт

### Требования

- Go 1.24.4 или новее
- Git

### Запуск примеров

#### 1. goBasics — Основы Go

```bash
cd easyGo/goBasics

# Установка зависимостей
go mod tidy

# Запуск
go run .

# Или сборка и запуск
go build -o goBasics .
./goBasics
```

**Что изучаем:**
- ✅ 5 способов объявления переменных
- ✅ Типы данных (int, float64, string, bool)
- ✅ Арифметические операции
- ✅ Работа со строками
- ✅ Константы

#### 2. goCalculator — Арифметика

```bash
cd easyGo/goCalculator

# Установка зависимостей
go mod tidy

# Запуск
go run .

# Или сборка и запуск
go build -o goCalculator .
./goCalculator
```

**Что изучаем:**
- ✅ Базовые арифметические операции
- ✅ Операции с плавающей точкой
- ✅ Возведение в степень
- ✅ Операции сравнения
- ✅ Инкремент и декремент

## 🎨 Примеры кода

### Объявление переменных

```go
// Способ 1: Короткое объявление
name := "Alex"
age := 25

// Способ 2: var с типом
var height int = 180

// Способ 3: var без типа
var city = "Moscow"
```

### Арифметика

```go
a, b := 20, 7

sum := a + b        // 27
diff := a - b       // 13
product := a * b    // 140
quotient := a / b   // 2 (целочисленное!)
remainder := a % b  // 6
```

### Константы

```go
const Pi = 3.14159
const MaxUsers = 1000
const AppName = "GoApp"
```
